### 良い練習をするために

良い練習をするために  
人は同じ動作の繰り返しは飽きてしまう生き物である（-->変化を付ける。ランダム性）  
人は集中が続かない生き物である（-->集中がない前提で時間を区切る、ポモドーロテクニック）  
人は失敗をする生き物である（-->失敗したときに気づく事が重要。振り返り）  
人は忘れっぽい生き物である（-->頭に頼らない。記録、構造化する） (edited)
